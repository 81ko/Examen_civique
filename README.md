# Examen civique CSP — Premium UI

Application statique FR/JP avec QCM, audio, vocabulaire, révision cochée et quiz de vocabulaire.

## Déploiement
Déposez le contenu du dossier sur GitHub puis connectez le dépôt à Vercel. Aucun build n'est nécessaire.

## Données locales
Les progrès, mots cochés et résultats sont enregistrés dans le navigateur avec localStorage.
